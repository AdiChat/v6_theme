The usage of {{#if author.*}} is no longer supported and should be replaced with {{#if primary_author.*}}or {{#if authors.[#].*}}.
Ghost allows multiple authors to be assigned to a post, so all helpers have been reworked to account for this.
Find more information about the {{authors}} helper here
Affected files
partials/post-card.hbs: Please remove or replace {{#if author.*}} from this template
The usage of {{error.code}} is deprecated and should be replaced with {{error.statusCode}}.
When in error context, e.g. in the error.hbs template, replace {{code}} with {{statusCode}}.
Find more information about the error.hbs template here.
Affected files
error-404.hbs: Please remove or replace {{error.code}} from this template
error.hbs: Please remove or replace {{error.code}} from this template
